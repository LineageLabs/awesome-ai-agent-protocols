# Contributing

Your contributions are welcome!

## Guidelines

- Search previous entries before making a new one to avoid duplicates.
- Make an individual pull request for each suggestion.
- Use the following format: `[Title](link) — Description.`
- New categories or improvements to the existing categorisation are welcome.
- Check your spelling and grammar.
- The pull request and commit should have a useful title.

## Adding a new entry

Entries should be:
- **Relevant** to AI agent protocols, infrastructure, identity, trust, payments, or governance.
- **Notable** — adopted by industry, backed by a standards body, or representing a novel contribution with a working implementation.
- **Well-documented** — with a publicly accessible specification or repository.
- **Active** — maintained within the last 6 months.

## What doesn't belong

- Agent frameworks and SDKs (LangChain, CrewAI, etc.) — there are dedicated lists for those.
- Foundation models (GPT, Claude, Gemini, etc.).
- Academic papers without a working implementation.
- Products that are pre-announcement or vaporware.

## Quality standards

- Link to the primary source (specification, official docs, or repo).
- Write a concise, factual description — no marketing language.
- Include a relevant stat where possible (adoption numbers, GitHub stars, transaction volume).
