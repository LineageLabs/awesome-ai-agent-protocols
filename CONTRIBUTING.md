# Contributing

Your contributions are welcome!

## Guidelines

- Search previous entries before making a new one to avoid duplicates.
- Make an individual pull request for each suggestion.
- Use the following format: `[Title](link) - Description.`
- New categories or improvements to the existing categorization are welcome.
- Check your spelling and grammar.
- The pull request and commit should have a useful title.

## Adding a new entry

Entries should be:
- **Relevant** to AI agent protocols, verification, trust, or identity.
- **Notable** -- published in a recognized venue, adopted by industry, or representing a novel contribution.
- **Well-documented** -- with a publicly accessible specification, paper, or repository.

## Quality standards

- Link to the primary source (specification, paper, or official repo).
- Include the proposer/author and year.
- Write a concise but informative description covering the key contribution.
